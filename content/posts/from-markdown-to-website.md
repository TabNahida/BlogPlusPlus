# One Command Workflow

Write markdown files under `content/posts` and run `blogpp build`.

## Supported markdown basics

- Headings
- Lists
- Links
- Inline code
- Fenced code blocks
- Blockquotes

> Keep content simple and portable.

## Plugin outputs

- `/tags/`
- `/archives/`
- `/feed.xml`
- `/search-index.json`
- `/sitemap.xml`
