# Welcome

BlogPlusPlus is a blog framework written in C++ with `xmake`.

## Why this project

- Fast static site generation
- Local preview server
- Familiar plugin architecture
- Theme system with reusable templates

## Code block demo

```cpp
#include <iostream>
int main() {
    std::cout << "Hello, BlogPlusPlus!" << std::endl;
    return 0;
}
```

## Next

Try these commands:

1. `xmake`
2. `xmake run blogpp build`
3. `xmake run blogpp serve 4000`
